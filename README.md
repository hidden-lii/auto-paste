# AUTO PASTE

## 安装步骤

### 1. 下载代码

首先，使用以下命令从代码库中获取项目代码：

```bash
git clone git@github.com:hidden-lii/auto-paste.git
```

### 2. 安装依赖

进入项目目录并执行以下命令来安装项目所需的依赖：

```bash
cd 项目目录
npm install
```

此命令将自动下载和安装项目所需的依赖项。

如果还没有安装过RUST，请访问 Rust 官方网站的[下载页面](https://www.rust-lang.org/tools/install)，按照步骤安装好RUST。

## 启动步骤

### 1. 运行开发环境

在安装项目依赖后，执行以下命令来启动开发环境：

```bash
npm run tauri dev
```
